# YouTube Comment Sentiment Analysis Web App

A web-based application built using **Python** and **Streamlit** that analyzes the sentiment of comments from a YouTube video. The app fetches comments using the **YouTube Data API**, processes them with **TextBlob**, and visualizes the sentiment distribution.

## Features

- Fetches comments from any public YouTube video
- Performs sentiment analysis on comments
- Categorizes comments as Positive, Neutral, or Negative
- Displays results using an interactive bar chart
- Simple and clean UI using Streamlit

- Python
- Streamlit

- TextBlob

## How the Application Works

1. Enter your **YouTube Data API Key**
2. Paste a **YouTube video URL**
3. The app extracts the video ID
4. Comments are fetched using YouTube Data API
5. Sentiment analysis is applied to each comment
6. Results are displayed visually
